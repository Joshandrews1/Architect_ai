# **App Name**: Architect AI

## Core Features:

- AI Growth Audit: A Vertex AI-powered chatbot that analyzes a provided URL or social media handle and provides a 'Status Analysis' of business growth opportunities using its tool for reasoning.
- Google Ecosystem Education: An interactive section explaining the integrations within the Google Ecosystem, including Vertex AI, Google Workspace, and Firestore.
- Automation Lab: A simulated 'Live Terminal' that displays real-time 'Status Updates' of simulated automation tasks, with data pulled from Firestore.
- Strategy Portal: A secure contact and booking form that writes data to a Firestore collection, triggering a Google Cloud Function to send an automated 'Growth Roadmap' email.
- Client Authentication: Enable secure authentication with Firebase Auth for access to personalized dashboards.

## Style Guidelines:

- Primary color: Cyber Yellow (#FFD300) for a high-tech industrial feel.
- Background color: Pure Black (#000000) for a minimalist, high-contrast aesthetic.
- Accent color: Electric Lime (#BFFF00) for active states and highlights, providing a visual cue for ongoing AI processing.
- Body and headline font: 'Space Grotesk' for a computerized, techy feel.
- High-tech, minimalist icons that fit the industrial UI, using glowing borders for active states.
- Grid-based layout for the Google Ecosystem section, using animated cards to showcase tool integrations.
- Subtle glowing borders for active AI processing states and smooth transitions between the 'Analysis' and 'Result' views.
